# 🛒 NextBuy — Customer Purchase Prediction

**Epitech Paris — B-DAT-201 Data Science Project**

## 🎯 What is NextBuy?

NextBuy predicts which products a supermarket customer 
is likely to buy next, based on their past order history. 
Built on Instacart-style grocery data, we trained and 
compared two machine learning models to achieve 
strong prediction accuracy.

---

## 🛠️ Tech Stack

| Tool | Usage |
|---|---|
| Python | Core language |
| Pandas / NumPy | Data manipulation |
| scikit-learn | ML models |
| Jupyter Notebook | Exploration & training |
| Matplotlib / Seaborn | Data visualization |

---

## 🤖 Models

| Model | Description |
|---|---|
| **Logistic Regression** | Baseline binary classifier |
| **Random Forest** | Ensemble model, best accuracy |

Both models predict the probability of a customer 
reordering a specific product in their next order.

---

## 📊 Dataset

Instacart-style grocery dataset containing:
- Customer order history
- Product catalog
- Order sequences and timing
- Reorder behavior

## 🚀 How to run

```bash
git clone git@github.com:Slacknsss/NextBuy-Machine-Learning.git
cd NextBuy-Machine-Learning
pip install -r requirements.txt

```

Open the main notebook and run all cells.
